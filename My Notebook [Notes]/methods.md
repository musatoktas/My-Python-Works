##Join Method
<h4>Description</h4>
<p>It helps to add string or integer to add between the list elements.</p>
<h4>Syntax</h4>
`listname.join(sequence)`
<h4>Usage</h4>
```
#!/usr/bin/python

s = "-";
seq = ("a", "b", "c"); # This is sequence of strings.
print s.join( seq )
```

##Append-Function
<h4>Description</h4>
<p>It works for append an elemnent to the list

